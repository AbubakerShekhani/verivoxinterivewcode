
Backend - NodeJS app for RESTful service
Frontend - index.html 


The index.html has JavaScript code which is being used to call REST API.


Instructions:
1. Run the backend using "node app.js" command
2. The server will be started on http://localhost:3005
3. Open front-end > index.html in browser
4. Enter the yearly consumption and press submit
5. It will show the comparison of the two products
